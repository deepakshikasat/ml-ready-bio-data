ML-Ready Biological Data Playbook

Bench to Bytes portfolio analysis report. Uses only public or synthetic data.

Hypothesis

If condition and batch are confounded, a model can appear accurate while learning technical artifacts instead of biology.

Data

Synthetic expression data generated to illustrate a common ML failure mode in biology: treatment labels are partially confounded with batch metadata.

Process

  1. Generated expression, condition, and batch metadata.
  2. Contrasted confounded, balanced-validation, and corrected scenarios.
  3. Wrote a practical ML-ready data guide.
  4. Created a wet-lab checklist for future ML projects.

Outputs

  • docs/ml_ready_data_playbook.md
  • templates/ml_project_checklist.md
  • outputs/batch_effect_demo.csv
  • figures/batch_effect_accuracy.svg

Interpretation

The accuracy drop under balanced validation is the key lesson: biological ML portfolios should show data strategy, not just model training.
