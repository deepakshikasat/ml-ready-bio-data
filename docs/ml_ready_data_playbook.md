# ML-Ready Biological Data Playbook

1. Define the prediction target before collecting data.
2. Treat assay labels as data products: versioned, audited, and linked to raw measurements.
3. Balance biological conditions across batches, operators, plates, and days.
4. Capture metadata that can explain technical variance.
5. Prefer standard identifiers such as Ensembl IDs, UniProt IDs, ChEMBL IDs, and sample ontologies.
6. Keep a simple baseline; if a rule beats ML, use the rule.
7. Reserve prospective validation data that mirrors the intended decision context.
