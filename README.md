# ML-Ready Biological Data Playbook

    A practical guide and synthetic demo showing how batch effects can dominate biological ML.

    ## Dataset

    Synthetic expression data created to demonstrate a common public ML pitfall in biology.

    ## Methods

    - Written playbook
- Batch-effect simulation
- Checklist template for wet-lab teams

    ## Reproduce

    ```bash
    python scripts/run_pipeline.py
    ```

    ## Outputs

    - `docs/ml_ready_data_playbook.md`
- `templates/ml_project_checklist.md`
- `outputs/batch_effect_demo.csv`
- `figures/batch_effect_accuracy.svg`

    ## Portfolio Note

    This repository uses public or synthetic demonstration data only. It is
    intended to show reproducible computational biology and AI/ML workflow
    design without relying on confidential or employer-owned material.
