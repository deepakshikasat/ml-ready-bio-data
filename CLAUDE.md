# ML-Ready Bio Data

This is a public portfolio project for the Bench to Bytes course. Use only
public datasets, synthetic demo data, or data generated by scripts in this
repository. Do not add employer-owned, patient-identifiable, or confidential
information.

Priorities:
- Make workflows reproducible from the command line.
- Save tables in `outputs/` and figures in `figures/`.
- Explain biological assumptions in plain language.
- Prefer simple baselines before complex models.
